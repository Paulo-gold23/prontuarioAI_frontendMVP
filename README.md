# prontuarioAI_frontendMVP
Projeto MVP de IA para transcrição automática de áudios clínicos em prontuários médicos estruturados. A solução reduz o tempo gasto com documentação, minimiza erros manuais e permite que profissionais da saúde foquem no atendimento ao paciente, mantendo privacidade e possibilidade de revisão humana.
